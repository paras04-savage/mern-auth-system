# MERN Authentication System

Full-stack authentication system built using:

- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- bcrypt password hashing

## Features

- User Registration
- Login Authentication
- JWT Token Protection
- Protected Routes
- Profile Page
- Logout Functionality

## Tech Stack

Frontend: React + Axios + React Router  
Backend: Express + MongoDB + JWT  

## Setup Instructions

### Backend

cd server  
npm install  
npm run dev  

### Frontend

cd client  
npm install  
npm start
